FoodMO: A Food Nutrient Analysis Application

FoodMO is a cutting-edge application designed to simplify food label interpretation using OCR and Random Forest. It helps users make healthier dietary choices by extracting text from food labels and analyzing the nutritional content of packaged foods. The system also provides personalized recommendations for healthier alternatives, making it a valuable tool for promoting well-informed eating habits.

  • OCR
  • Random Forest
  • Cosine Similarity
Physics IRL

Physics IRL is an augmented reality (AR) application that simplifies physics concepts like frictional and gravitational forces. Using camera permissions, the app detects horizontal planes via the AR Plane Manager. Plane is shown as a dotted mesh on which users can place 3D objects on the plane to study gravitaionl force. To explore frictional force, Unity's AR Tracked Image Manager tracks predefined images in a Tracked Images Library, enabling precise placement of 3D models. The implementation has been done using custom C# scripts. With realistic textures and interactive simulations, Physics IRL provides an immersive experience, and making physics easy to imagine and understand.

  • Unity
  • C#
  • Object Detection
ToDo List: Cloud-Based Multi-Device To-Do List Application

This project utilizes AWS EC2, Security Groups, and Django to create a scalable, cloud-based to-do list application with automatic scaling, reducing operational overhead. MyReckList

This website is designed to recommend movies and books tailored to user preferences based on their ratings. By leveraging the Alternating Least Squares (ALS) algorithm, it implements collaborative filtering to analyze user behavior and predict personalized recommendations. The system processes user data to identify patterns and suggest the top 10 most relevant movies and books, ensuring a highly accurate and user-centric experience. Whether a user is exploring new genres or looking for similar content, the platform delivers recommendations that align with their tastes, enhancing discovery and engagement. This approach optimizes content curation, offering users a seamless, enjoyable experience.

  • Website
  • Collaborative Filtering
Umanità

Umanità is a website designed to help NGOs create and manage their own websites, providing them with a virtual presence in the world. Built using HTML, CSS, and JS, Umanità allows organizations to design customizable websites that reflect their unique missions and goals. The platform provides tools for easy content management, allowing NGOs to display information, share updates, and engage with their audience seamlessly. Key features include donation integrations, event listings, and volunteer sign-up forms, to support the NGOs' objectives. Umanità simplifies website creation, enabling organizations to focus on their core mission while enhancing their online reach and impact.

  • Website

ECG Report Analysis Leveraging Generative AI

This tool utilizes CNNs to analyze and classify ECG reports with 92% accuracy, identifying multiple heart conditions. It provides users with a simplified analysis tailored to their profile. Integrating Generative AI, it aims to enable users to interact with an LLM model for better understanding, suggesting lifestyle changes, and locating nearby cardiologists. Additionally, the tool plans to offer insights into the heart's potential future condition based on current trends. While ECG classification is complete, further development is focused on enhancing the LLM-powered interaction and predictive insights, making it a comprehensive solution for heart health monitoring and guidance.

  • CNN Models
  • Generative AI
